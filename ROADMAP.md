# Phazer Labs Roadmap

## Planned Features

### Homepage — Recent YouTube Videos Feed
Add a "Recent Videos" section to the homepage below the "What I Do" pillars section that pulls the latest YouTube videos from the [@phazerlabs](https://youtube.com/@PhazerLabs) channel and displays them in a feed. Considerations:
- YouTube Data API v3 with an API key (most reliable, requires Google API key setup)
- RSS-to-JSON proxy service (no API key, but depends on third-party service)
- Static embeds (simplest, but requires manual updates)
